## HR Analytics Dashboard

**Files:** `HR_Analytics.twbx` (recommended) or `HR_Analytics.twb`  
**Live Dashboard:** [View on Tableau Public](https://public.tableau.com/views/TableauDashFinal/Dashboard1)  
**Dataset:** `HR_Employees.xlsx` - simulated HR data for ~500 employees across 6 departments

---

### Overview

An HR workforce demographics dashboard built in Tableau covering headcount, gender distribution, age distribution, salary, and education level across departments.

---

### Dashboard Sheets

| Sheet | Description |
|-------|-------------|
| Q1 | Dataset overview - employee count, dimensions vs. measures |
| Q2 | KPI summary - average age, salary, and tenure |
| Q3 | Employee count by department |
| Q4 | Gender distribution |
| Q5 | Employee age distribution |
| Q6 | Employee count by education level |

---

### How to Open

1. Download `HR_Analytics.twbx` and open in Tableau Desktop 2026.2 or later -- no additional files needed
2. Or download `HR_Analytics.twb` and `HR_Employees.xlsx` and keep them in the same folder

> Or view the published version via the Tableau Public link above.
