## Sales Analytics Dashboard

**File:** `Sales_Dashboard.pbix`  
**Live Dashboard:** [View on Power BI Service](https://app.powerbi.com/groups/me/reports/3877b6c9-a46c-4039-94c4-c844c0098048/cb02bfa1e43cdcb92385?experience=power-bi)  
**Dataset:** `sales_data.csv` - 1,000 sales transactions across reps, regions, product categories, and channels

---

### Overview

An interactive sales dashboard built in Power BI Desktop. Features slicers for region, product category, and date range that cross-filter all visuals in real time.

---

### Dashboard Visuals

| Visual | Description |
|--------|-------------|
| Slicers | Filter by region, product category, and date range |
| Transaction Table | Top transactions by sales amount with rep, region, and category breakdown |
| Bar Chart | Sales amount by channel - Retail vs. Online |

---

### How to Open

1. Download `Sales_Dashboard.pbix` and open in [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. If prompted for credentials, dismiss - data is embedded in the file

> Or view the live published version via the link above.
