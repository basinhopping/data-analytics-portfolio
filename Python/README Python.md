## Student Exam Performance Analysis

**Dataset:** `StudentsPerformance.csv` - 1,000 students with math, reading, and writing scores alongside background attributes  
**Source:** [Kaggle - Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)

---

### Files

| File | Description |
|------|-------------|
| `Midterm_Assessment.ipynb` | Individual six-part analysis covering EDA, visualization critique, technical report, and ethics reflection |
| `Student_Performance_EDA.ipynb` | Group project covering six targeted visualizations across demographic and performance factors |
| `StudentsPerformance.csv` | Dataset used by both notebooks |

---

### How to Run

1. Place `StudentsPerformance.csv` in the same folder as the notebooks
2. Install dependencies:
   ```bash
   pip install pandas seaborn matplotlib jupyter
   ```
3. Open either notebook in Jupyter and run all cells top to bottom
