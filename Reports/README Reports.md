### Overview

A formal write-up of the student exam performance analysis covering purpose, exploratory data analysis, a technical report with methods and results, an ethics reflection, and challenge questions. Compiled in Overleaf.

---

### How to Compile the LaTeX File

1. Go to [Overleaf](https://www.overleaf.com) and create a new project
2. Upload `Student_Performance_Report.tex` and the four chart images:
   - `chart1_test_prep.png`
   - `chart2_distribution.png`
   - `chart3_reading_writing.png`
   - `chart4_lunch.png`
3. Click Recompile
